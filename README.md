# ✈️  Smart Carrier
스마트 캐리어

## 💻 프로젝트 소개 (Introduction)
아두이노와 앱을 연동하여 만든 스마트캐리어 어플입니다.

## 💡 주요 기능
#### 메인화면 
- 블루투스 연결
- 디바이스 선택
- 자동 검색
- 앱 정보
#### 캐리어 찾기
- 도난방지 알림
- 도난방지 소리 ON/OFF
#### 캐리어 무게 측정
- 무게 설정 및 값 저장
- 무게 측정
#### 백드랍 모드
- 백드랍 모드 연결
- 시각 설정
#### 캐리어 정보 
- RSSI 측정값, 자동검색 사용여부, 도난방지 작동여부, 블루투스 연결
- 배터리 잔량 확인
